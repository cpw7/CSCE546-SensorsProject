# Tilt Balance Lab – CSCE 546 Sensors Project

## Purpose
This app uses phone sensors to create an interactive tilt-control balance game and posture/session report.

## Sensors Used
- Accelerometer: controls the blue ball and estimates tilt.
- Gyroscope: detects rotation/jitter and affects stability scoring.
- Light sensor: changes the environment/difficulty label based on brightness.

## Features
- Tilt the phone to move the blue ball.
- Keep the ball inside the green bullseye to build score.
- Stability score rewards smooth control.
- Posture tracker classifies flat, vertical, angled, and active movement.
- Session report tracks time spent in posture states.
- Light sensor changes the environment label.
- Tap reset to restart and re-center the game.
