# Sensors Project App

CSCE 546 Mobile Application Development

This Android/Kotlin project uses two non-GPS sensors:

- Accelerometer
- Magnetometer

The app displays live sensor values, estimates tilt, estimates compass direction, and includes a simple shake counter. GPS is not used in this project.

## Run

Open the inner `SensorsProjectApp` folder in Android Studio, wait for Gradle sync, select an emulator, and click Run.

## Emulator testing

Use the emulator three-dot menu, open Virtual sensors, and change the device position to update the accelerometer and magnetometer readings.
